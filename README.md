# etown-cs-homelab.github.io
GitHub-hosted documentation for our Homelab project: [https://etown-cs-homelab.github.io/](https://etown-cs-homelab.github.io/)
### Local Deployment
- Install [Node.js](nodejs.org)
- Clone the blog's [repository](https://github.com/etown-cs-homelab/etown-cs-homelab.github.io)
- Run the following:
```
npm install
npm start
```
There are no additional APIs that we're using to run this.
